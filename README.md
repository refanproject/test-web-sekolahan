# SMK Negeri 1 Maja - Website test by refan

Complete clone of the SMK Negeri 1 Maja official website with all pages, responsive design, and modern UI.

## Features
- Fully responsive design
- All pages from original website
- Modern and clean UI
- Interactive navigation
- News/Berita section
- Program Keahlian pages
- Profile pages

## How to Run
1. Install dependencies: `npm install`
2. Start server: `npm start`
3. Or simply open `index.html` in your browser

## Structure
- `/` - Main homepage
- `/profil/` - Profile pages
- `/kurikulum/` - Curriculum pages
- `/program/` - Program pages
- `/berita/` - News pages
