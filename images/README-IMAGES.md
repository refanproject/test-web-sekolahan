# Instruksi Gambar Background

## File yang Diperlukan

### 1. page-header-bg.jpg
**Lokasi:** `images/page-header-bg.jpg`

**Deskripsi:** 
File ini adalah clone/duplikat dari `sekolah.png` yang digunakan di hero section, tetapi dalam format JPG.

**Cara Membuat:**
1. Buka file `images/sekolah.png`
2. Save As atau Export sebagai `page-header-bg.jpg`
3. Simpan di folder `images/`
4. Pastikan kualitas JPG di atas 80% untuk hasil yang baik

**Penggunaan:**
File ini akan digunakan sebagai background untuk:
- Halaman Profil (Visi Misi, Sejarah, Kepala Sekolah, dll)
- Halaman Kontak
- Halaman Kurikulum (RPL, SIJA, Kalender Akademik, dll)
- Halaman Berita
- Halaman Program

**Spesifikasi Rekomendasi:**
- Format: JPG
- Resolusi minimal: 1920x1080px
- Ukuran file: < 500KB (untuk performa optimal)
- Kualitas: 80-90%

**Efek yang Diterapkan:**
- Linear gradient overlay (hijau primary ke secondary)
- Dark overlay (opacity 30%)
- Background blend mode: overlay
- Background size: cover
- Background position: center

## Alternatif Jika File Tidak Ada

Jika file `sekolah.png` tidak tersedia atau Anda ingin menggunakan gambar lain:

1. Gunakan gambar sekolah yang representatif
2. Pastikan gambar memiliki resolusi tinggi
3. Gambar sebaiknya menampilkan:
   - Gedung sekolah
   - Suasana sekolah
   - Atau elemen yang mewakili pendidikan

## Catatan Teknis

Background ini sudah dikonfigurasi dengan:
- Gradient overlay otomatis menggunakan warna primary (#16a34a) dan secondary (#22c55e)
- Dark overlay untuk meningkatkan kontras teks
- Responsive dan akan menyesuaikan dengan ukuran layar
- Blend mode overlay untuk efek visual yang menarik
