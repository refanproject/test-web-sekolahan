Tentang Program Keahlian SIJA

Program Keahlian Sistem Informasi Jaringan dan Aplikasi (SIJA) adalah program pendidikan yang mempersiapkan siswa untuk menjadi Network Administrator, System Administrator, dan IT Infrastructure Specialist yang kompeten. Program ini dirancang untuk menghasilkan lulusan yang mampu merancang, mengimplementasikan, dan mengelola infrastruktur jaringan komputer serta sistem informasi.

Kompetensi Keahlian

Siswa program keahlian SIJA akan mempelajari berbagai kompetensi, antara lain:

  • Dasar-dasar Jaringan Komputer (OSI Layer, TCP/IP)
  • Routing dan Switching (Cisco, MikroTik)
  • Network Security dan Firewall
  • Wireless Network (WiFi, WLAN)
  • Server Administration (Windows Server, Linux Server)
  • Cloud Computing (AWS, Azure, Google Cloud)
  • Virtualization (VMware, VirtualBox, Docker)
  • Network Monitoring dan Troubleshooting
  • VoIP dan Video Conference
  • IoT (Internet of Things)
  • Database Administration
  • Web Server Configuration (Apache, Nginx)

Sertifikasi

Program keahlian SIJA menyiapkan siswa untuk mendapatkan sertifikasi internasional:

  • Cisco Certified Network Associate (CCNA)
  • MikroTik Certified Network Associate (MTCNA)
  • CompTIA Network+
  • Linux Professional Institute Certification (LPIC)
  • Microsoft Certified Solutions Associate (MCSA)

Fasilitas

Program keahlian SIJA didukung oleh fasilitas yang lengkap dan modern:

  • Laboratorium Jaringan dengan perangkat Cisco dan MikroTik
  • Server Room dengan berbagai jenis server
  • Perangkat networking (Router, Switch, Access Point)
  • Tools untuk network monitoring dan troubleshooting
  • Akses ke platform cloud computing
  • Simulator jaringan (Packet Tracer, GNS3)

Prospek Karir

Lulusan program keahlian SIJA memiliki prospek karir yang sangat luas:

  • Network Administrator
  • System Administrator
  • Network Engineer
  • IT Support Specialist
  • Cloud Engineer
  • Network Security Specialist
  • VoIP Engineer
  • Data Center Technician
  • IT Consultant
  • Entrepreneur di bidang IT Infrastructure

Kerjasama Industri

Program keahlian SIJA menjalin kerjasama dengan berbagai perusahaan untuk memberikan pengalaman praktik kerja:

  • Internet Service Provider (ISP)
  • Perusahaan Telekomunikasi
  • Data Center Companies
  • IT Consulting Firms
  • Bank dan Institusi Keuangan
  • Perusahaan Multinasional

Prestasi

Siswa program keahlian SIJA telah meraih berbagai prestasi:

  • Juara LKS bidang IT Network Systems Administration
  • Juara kompetisi jaringan tingkat nasional
  • Sertifikasi internasional CCNA dan MTCNA
  • Pemenang berbagai hackathon dan CTF competition
