Tentang Program Keahlian Rekayasa Perangkat Lunak

Program Keahlian Rekayasa Perangkat Lunak (RPL) adalah program pendidikan yang mempersiapkan siswa untuk menjadi programmer, software developer, dan IT professional yang kompeten. Program ini dirancang untuk menghasilkan lulusan yang mampu merancang, mengembangkan, dan memelihara aplikasi perangkat lunak sesuai dengan kebutuhan industri.

Kompetensi Keahlian

Siswa program keahlian RPL akan mempelajari berbagai kompetensi, antara lain:

  • Pemrograman Dasar (Algorithm, Flowchart, Pseudocode)
  • Pemrograman Berorientasi Objek (Java, C++, Python)
  • Pemrograman Web (HTML, CSS, JavaScript, PHP)
  • Pemrograman Mobile (Android, iOS, Flutter)
  • Database Management (MySQL, PostgreSQL, MongoDB)
  • Framework Development (Laravel, React, Vue.js, Node.js)
  • Software Engineering (SDLC, Agile, Scrum)
  • UI/UX Design
  • Version Control (Git, GitHub)
  • Testing dan Debugging
  • DevOps dan Cloud Computing
  • Cyber Security Basics

Fasilitas

Program keahlian RPL didukung oleh fasilitas yang lengkap dan modern:

  • Laboratorium Komputer dengan spesifikasi tinggi
  • Software Development Tools (IDE, Compiler, Framework)
  • Server untuk praktik deployment aplikasi
  • Akses ke platform pembelajaran online
  • Lisensi software profesional
  • Ruang kolaborasi dan project-based learning

Prospek Karir

Lulusan program keahlian RPL memiliki prospek karir yang sangat luas, antara lain:

  • Software Developer / Programmer
  • Web Developer (Frontend/Backend/Fullstack)
  • Mobile App Developer
  • Database Administrator
  • System Analyst
  • Quality Assurance / Software Tester
  • UI/UX Designer
  • DevOps Engineer
  • IT Consultant
  • Entrepreneur di bidang IT

Kerjasama Industri

Program keahlian RPL menjalin kerjasama dengan berbagai perusahaan teknologi terkemuka untuk memberikan pengalaman praktik kerja industri (Prakerin) dan peluang kerja bagi lulusan, antara lain:

  • Perusahaan Software House
  • Startup Technology
  • Perusahaan IT Consulting
  • E-commerce Companies
  • Digital Agency
  • Bank dan Fintech

Prestasi

Siswa program keahlian RPL telah meraih berbagai prestasi, baik di tingkat regional, nasional, maupun internasional:

  • Juara Lomba Kompetensi Siswa (LKS) bidang Software Engineering
  • Juara Hackathon dan Coding Competition
  • Pemenang berbagai kompetisi aplikasi mobile dan web
  • Sertifikasi kompetensi internasional
